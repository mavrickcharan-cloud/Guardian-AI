# 🛡️ Guardian AI

> AI-powered personal safety application designed to help people during emergencies using artificial intelligence and smart safety features.

## 📖 About

Guardian AI is an Android application focused on improving personal safety through intelligent technology. The project aims to provide fast emergency assistance, smart alerts, and AI-powered safety tools for everyday situations.

## ✨ Features

- 🚨 Emergency SOS
- 🤖 AI Emergency Assistant
- 📍 Live Location Sharing (Planned)
- 🎙️ Voice-Activated SOS (Planned)
- 🛡️ Smart Safety Monitoring
- 🔔 Instant Emergency Alerts
- 🌐 Offline Safety Support (Planned)

## 🛠️ Tech Stack

- Google AI Studio
- Gemini AI
- Android Development
- Firebase (Planned)
- Python
- Git & GitHub
